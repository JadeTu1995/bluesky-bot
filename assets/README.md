# Assets 目录说明

本目录用于存放TOC图片文件。

## 使用方法

1. 将TOC图片文件放在此目录下
2. 在Google Sheets的"TOC图片"列填写文件名

## 示例

```
assets/
├── README.md              (本文件)
├── toc_image1.jpg         (TOC图片1)
├── toc_image2.png         (TOC图片2)
└── another_toc.jpg        (另一个TOC图片)
```

## 注意事项

- 图片大小建议在2MB以下
- 支持的格式：JPG, PNG, GIF, WebP
- 文件名可以使用中文、英文、数字

## 定时任务

如果使用定时任务自动发送推文，请将所有需要的TOC图片放在此目录下。
